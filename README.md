# 📝 TodoApp

A modern and intuitive **Todo application** developed using **Flutter** and **Dart**. The application helps users organize their daily tasks with a simple, responsive, and user-friendly interface. Task data is stored using **Firebase Cloud Firestore**, allowing real-time updates.

---

## ✨ Features

* ➕ Add new tasks
* 📝 Edit existing tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* ☁️ Real-time data synchronization with Firebase Firestore
* 📱 Responsive and clean Material Design interface
* ⚡ Smooth and fast user experience

---

## 🛠️ Built With

* Flutter
* Dart
* Firebase Cloud Firestore
* Material Design

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/sudesgrc/TodoApp.git
```

Navigate to the project directory:

```bash
cd TodoApp
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```
